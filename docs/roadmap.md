# Roadmap

This document outlines the planned development phases for the BizzConnect application.

## Phase 1: Foundation and Core Models

- [x] Project setup and initial documentation.
- [ ] Expand the Prisma schema with core models for `User`, `SupplierProfile`, `Category`, `Subcategory`, and `Product`.
- [ ] Implement user authentication (registration and login).
- [ ] Seed the database with initial data.

## Phase 2: Supplier and Product Management

- [ ] Implement supplier profile creation and management.
- [ ] Implement product creation, editing, and deletion.
- [ ] Create public-facing pages for products and suppliers.

## Phase 3: RFQ and Lead Management

- [ ] Implement the RFQ submission process for buyers.
- [ ] Create dashboards for buyers and suppliers to manage leads.
- [ ] Implement a basic messaging system for leads.

## Phase 4: Search and Discovery

- [ ] Implement a global search for products and suppliers.
- [ ] Add filtering and sorting capabilities.

## Phase 5: Monetization and Subscriptions

- [ ] Implement subscription plans for suppliers.
- [ ] Create a basic UI for plan upgrades.

## Phase 6: Reviews and Admin Panel

- [ ] Implement a review and rating system for suppliers.
- [ ] Build an admin panel for user and content management.
