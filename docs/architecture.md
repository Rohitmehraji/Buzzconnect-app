# Architecture

This document outlines the high-level architecture of the BizzConnect application.

## Tech Stack

- **Frontend:** Next.js, React, TypeScript, Tailwind CSS
- **Backend:** Next.js API Routes, Prisma
- **Database:** PostgreSQL

## Overview

The application is a monolith built with Next.js. The frontend is composed of React components written in TypeScript and styled with Tailwind CSS. The backend is implemented using Next.js API Routes, which handle business logic and database interactions through Prisma.
