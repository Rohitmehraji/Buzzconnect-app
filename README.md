# BizzConnect

BizzConnect is a B2B marketplace platform inspired by IndiaMART, built with Next.js, Prisma, and PostgreSQL.

## Getting Started

First, set up your environment variables by creating a `.env` file in the root of the project:

```
DATABASE_URL="postgresql://user:password@host:port/database"
```

Next, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Learn More

To learn more about the project's architecture, data model, and roadmap, please refer to the documents in the `docs` directory.
