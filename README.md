# BizzConnect

BizzConnect is a B2B marketplace that connects buyers and suppliers, inspired by the core features of IndiaMART.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Frontend:** [Next.js](https://nextjs.org/), [React](https://react.dev/), [TypeScript](https://www.typescriptlang.org/), [Tailwind CSS](https://tailwindcss.com/)
- **Backend:** Next.js API Routes, [Prisma](https://www.prisma.io/), [PostgreSQL](https://www.postgresql.org/)

## Project Structure

- `pages/`: Contains the frontend pages of the application.
- `pages/api/`: Contains the backend API routes.
- `prisma/`: Contains the Prisma schema and migrations.
- `lib/`: Contains shared library code, such as Prisma client instances and hooks.
- `components/`: Contains reusable React components.
- `styles/`: Contains global CSS styles.
- `public/`: Contains static assets.
- `docs/`: Contains project documentation.
